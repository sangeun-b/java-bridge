## 🚀 기능 목록
### 1. 출력
- [x] 게임 시작 메세지
- [ ] 게임 종료 문구
2. 입력
- [x] 다리 길이 입력
- [ ] 이동할 칸 입력
- [ ] 게임 재시작/종료 여부 입력
3. 게임
- [x] 입력 받은 다리 길이로 랜덤 다리 생성
- [ ] 이동할 수 있는 칸인지 확인
- [ ] 게임 성공/실패 여부 판단
- [ ] 게임 재시작
- [ ] 총 시도 횟수

## 🤔 예외 처리
### ⚠️IllegalArgumentException 발생 후, "[ERROR]"로 시작하는 에러 메시지 출력 후 종료
- 다리 길이
    - [ ] 다리 길이가 3이상 20이하가 아닌 경우
    - [ ] 다리 길이 입력 값이 숫자가 아닌 경우
- 이동 할 칸
    - [ ] 이동 입력 값이 U or D가 아닌 경우
- 게임 재시작
    - [ ] 게임 재시작 값이 R or Q가 아닌 경우
